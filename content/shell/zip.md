# zip 和 unzip

压缩
```shell
zip -r filename.zip filesdir

zip -r -S filename.zip filesdir #包含系统和隐藏文件

# or zip multi-files to one zip file
zip -r filename.zip file1 filesdir filesdir2
```

解压

```shell
unzip filename.zip
```
